# Parallel downloader

Parallel downloader is a tool for downloading files in parallel.

## Usage

```bash
go run cmd/main.go \
  -url https://d37ci6vzurychx.cloudfront.net/trip-data/yellow_tripdata_2018-05.parquet \
  -output yellow_tripdata_2018-05.parquet
```
